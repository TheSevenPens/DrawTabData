# DATA LAYOUT

JSON files are organized by entity type and partitioned by brand.

	/data
		/tablets
			HUION-tablets.json
			WACOM-tablets.json
			XENCELABS-tablets.json
			XPPEN-tablets.json
		/pens
			WACOM-pens.json
		/pen-families
			WACOM-pen-families.json
		/tablet-families
			WACOM-tablet-families.json
		/pen-compat
			WACOM-pen-compat.json
		/drivers
			WACOM-drivers.json

Each file wraps its records in a top-level key:
- tablets: { "DrawingTablets": [...] }
- pens: { "Pens": [...] }
- pen-families: { "PenFamilies": [...] }
- tablet-families: { "TabletFamilies": [...] }
- pen-compat: { "PenCompat": [...] }
- drivers: { "Drivers": [...] }

File naming convention: BRAND-entitytype.json (e.g. WACOM-tablets.json)
