# FUTURES

## DATA

### Add nibs dataset
Nib types, materials, compatibility with pens.

### Expand pen data
Add fields for technology, buttons, pressure levels, tilt support, weight.

### Expand pen/tablet family data
Add more metadata to families as the concept grows.

### Add pen-compat for other brands
Currently only Wacom. HUION, XPPEN, XENCELABS need compatibility data.

### Add drivers for other brands
Currently only Wacom.

### Pressure response data
Pressure response curves per pen/tablet combination. Different data shape
(time series per device pair) needing its own loader.

### Additional computed fields

- Pixel count (width x height)
- Aspect ratio

### Data quality for all entity types
Currently only validates tablets. Extend to pens, drivers, families, compat.
